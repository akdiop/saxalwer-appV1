## 🐛 Points d'Attention

### LocalStorage
- Les rendez-vous sont stockés localement
- Synchronisation cloud non implémentée (prototype)
- Données effacées si cache navigateur vidé

### Données de Cycle
- Les statistiques dépendent des données du tracker
- Encourager l'utilisatrice à remplir régulièrement

### Performance
- Recharts charge ~60KB (acceptable)
- Graphiques optimisés avec ResponsiveContainer
